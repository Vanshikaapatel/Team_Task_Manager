# 🧠 Team Task Manager

A simple full-stack web application built to manage tasks efficiently within a team.
This project focuses on core functionalities like authentication, task management, and basic dashboard insights.

---

## 🚀 Features

### 🔐 Authentication

* User Signup and Login
* JWT-based authentication
* Secure access to user-specific data

---

### 📋 Task Management

* Create new tasks
* View all tasks
* Update task status

  * To Do → In Progress → Done
* Delete tasks
* Assign tasks to users

---

### 📊 Dashboard

* Total number of tasks
* Tasks by status (Done / Pending)
* Overdue task tracking

---

### 📁 Project Management (Basic)

* Create projects
* Associate tasks with projects

---

## ⚙️ Installation (Run Locally)

1. Clone the repository:

```bash
git clone https://github.com/Vanshikaapatel/Team_Task_Manager.git
```

2. Go to project folder:

```bash
cd Team_Task_Manager
```

3. Install dependencies:

```bash
npm install
```

4. Create `.env` file:

```env
MONGO_URI=your_Local_mongodb_connection_string
JWT_SECRET=your_secret_key
```

5. Run the server:

```bash
node server.js
```
